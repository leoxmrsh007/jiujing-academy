## 平台功能架构

### 1. 经典原文层
- **多版本对照**
  - 原文（繁体/简体）
  - 白话译文
  - 英译本（Legge/Waley等）
  - 日译本、韩译本
  
- **章节导航**
  - 目录树结构
  - 关键词检索
  - 上下文关联

### 2. 研究课程层
- **基础课程**
  - 经典导读
  - 字词训诂
  - 篇章结构
  
- **进阶课程**
  - 思想阐释
  - 历史背景
  - 学术史梳理
  
- **专题研究**
  - 跨经典比较
  - 中外哲学对话
  - 现代应用

### 3. 二阶心智训练
- **苏格拉底式提问**
  - 自动生成的反思问题
  - 对话式学习路径
  
- **个人修学日志**
  - 学习轨迹记录
  - 心得体会整理
  - 阶段性总结

### 4. 中外对比研究
- **东亚圈**
  - 中国（儒释道）
  - 日本（神道、禅宗）
  - 韩国（性理学）
  - 越南（儒学）
  
- **西方哲学**
  - 古希腊（柏拉图、亚里士多德）
  - 德国古典（康德、黑格尔）
  - 存在主义（海德格尔）
  - 过程哲学（怀特海）

## 核心特色：二阶心智教学法

### 不只是知识传授，更是思维训练

**传统教学：**
```
经典原文 → 注释讲解 → 背诵记忆 → 考试检验
```

**二阶心智教学：**
```
经典原文 → 一阶理解 → 二阶反思 → 意义生成 → 实践应用 → 回写提升
           ↓
      「这段话对你意味着什么？」
      「如何应用到你的生活中？」
      「与其他思想如何对话？」
```
